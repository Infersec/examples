# Infersec Examples

Public example projects showing how to integrate with the [Infersec](https://infersec.ai) platform and its OpenAI-compatible API.

Each example is self-contained. Clone this repository, run `npm install` inside the example directory, and follow that example's README.

## Examples

| Example | Stack | Description |
| --- | --- | --- |
| [`mastra-web-research`](./mastra-web-research) | Mastra, TypeScript | A web research agent powered by a model hosted on Infersec. |

## Prerequisites

Most examples need an Infersec account with a hosted model, an enabled **inference endpoint**, and an **API key**. Copy each example's `.env.example` to `.env` and fill in your details.
